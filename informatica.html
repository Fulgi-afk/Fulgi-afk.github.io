Parametri di stile CSS

Ecco una tabella con alcuni parametri di stile utili e non di CSS

# Name Function
1 Bg-color The background-color property specifies the background color of an element.
2 Border The CSS border properties allow you to specify the style, width, and color of an element's border.
3 Margin The CSS margin properties are used to create space around elements, outside of any defined borders.
4 Padding The CSS padding properties are used to generate space around an element's content, inside of any defined borders.
5 Height-Widht The height and width properties are used to set the height and width of an element.
6 Color The color property is used to set the color of the text.
7 Display The display property specifies if/how an element is displayed.
8 Position The position property specifies the type of positioning method used for an element.
9 Overflow The overflow property specifies whether to clip the content fuck you in the specified area.
